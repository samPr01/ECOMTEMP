## Quick Start

1. Install all dependencies:
   ```bash
   npm run install-all
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser to:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:5000
